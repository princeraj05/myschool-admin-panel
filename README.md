# MySchool Admin Panel

A full-stack School Management System built using React, Node.js, Express, and MongoDB.

## Features

* Admin Dashboard
* Teacher Panel
* Student Panel
* Class Management
* Attendance System
* Exam Scheduling
* Real-time updates with Socket.io

## Tech Stack

Frontend:

* React
* Vite
* TailwindCSS

Backend:

* Node.js
* Express
* MongoDB
* Socket.io

## Installation

### Backend

```
cd backend
npm install
npm start
```

### Frontend

```
cd frontend
npm install
npm run dev
```
